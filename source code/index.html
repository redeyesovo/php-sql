关于青木原

青木原树海位于日本富士山西北侧山麓，横跨山梨县富士河口湖町与鸣泽村，周围有西湖、精进湖与本栖湖等堰塞湖。青木原为熔岩台地地形，标高约介于920米～1300米之间，面积约有3000公顷。查看图片.

青木原位于富士箱根伊豆国立公园中，保有大面积原生林。青木原邻近富士山、富士五湖，国道139号从中穿过，森林浴步道及露营营地等公共设施整备齐全。

由于日本作者松本清张所著‘波の塔’一书与相关影音衍生之风行流传，青木原树海逐渐成为日本的自杀胜地。

青木原树海地下有磁铁矿能导致指南针失灵 青木原树海之地质为熔岩冷却后所形成具磁性的玄武岩，在具有大量、大范围玄武岩之地区确实能使指南针偏移。于青木原树海指南针约会偏移1～2度，因此日本自卫队常于此处进行定位训练。

在青木原树海中，GPS不能运作 该情形与地下磁矿无关。实际上是由于青木原树海林叶茂密，因此较为低阶的GPS系统易因讯号接收不到而无法使用。近年由于基地台增置及GPS技术提升， GPS运作情形有大幅提升。

因为地下磁矿会干扰雷达，因此禁止民用机飞经上空 日本青木原树海地区确实禁止民用机飞经该地区上空。然而这是因为一方面该地区部分为军方用地，鉴于情资保密而禁止。另一方面是因该地区上空常形成乱流，鉴于飞航安全而禁止。
